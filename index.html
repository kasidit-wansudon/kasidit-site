claude code plugin

Mindful AI coding framework.
Discipline over cleverness.

Kasidit forces structure — two-tier knowledge, checklists, and verification loops — so any model tier (Opus, Sonnet, Haiku) stops guessing and starts shipping. Runtime is the judge.

Get started View on GitHub

Install

claude code >= 1.0 
/plugin marketplace add kasidit-wansudon/kasidit
/plugin install kasidit

Paste both lines into Claude Code. No build step. No npm. No config.

Why Kasidit

Gravity Pattern

Two-tier knowledge: a compact Centerlite always in context, a deep Dcenterlite loaded on demand. Small context, full recall.

Checklists, not guesses

Audits run against explicit checklists — security, review, UI. The checklist does the thinking; the model executes.

Tier-aware

Adapts prompts and verification depth to Opus, Sonnet, or Haiku. Weaker reasoning is compensated by more structure, not more tokens.

Commands

/kasi-init

Scaffold .kasidit/ knowledge into a new project.

/kasi-fix

Conservative bug-fix mission with verification loops.

/kasi-review

Code review driven by the Kasidit checklist.

/kasi-security

Security audit with explicit threat checklist.

/kasi-ui

UI and CSS fix mission with Override Mode.

/kasi-cascade

Tier cascade: Opus plans, Sonnet works, Haiku greps.
