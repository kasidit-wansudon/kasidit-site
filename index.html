How it works

A mission moves through three gates. Each gate refuses if the prior is underspecified.

  1. 01

    Narrow the mission

    Vague asks get refused. Pick from numbered options until the target is verifiable.

  2. 02

    Dispatch specialists

    sudo <mission> fans out to 6 parallel specialists. Each gets isolated context and a brief.

  3. 03

    Synthesize with confidence

    Findings tagged [high/medium/low/unsure]. [unsure] never silently guessed.

Install

Run these two lines in Claude Code. First time only.

/plugin marketplace add kasidit-wansudon/kasidit
/plugin install kasidit

Update

Already installed. Pull the latest release.

/plugin marketplace update kasidit

Reload the session to pick up new commands. New in the latest release appears on the releases page.

Prompt log hook (optional)

Capture every prompt to ~/.claude/skills/kasidit/center/logs/. Hook ships with the plugin; installs at user scope.

cp ~/.claude/plugins/marketplaces/kasidit/plugins/kasidit/hooks/kasidit-log.{sh,py} ~/.claude/hooks/ && chmod +x ~/.claude/hooks/kasidit-log.*

Then register the UserPromptSubmit hook in ~/.claude/settings.json — full steps in the Installation wiki.
